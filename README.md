# JoyHub IM Ticket Demo

This repository is a static, clickable demo for the JoyHub IM ticket workspace.

It does not include production backend code, login flow, API credentials, autosave scripts, QA screenshots, or local working notes.

## Pages

- `index.html`: main interactive demo.
- `src/pages/user-chat.html`: standalone user chat demo page.
